## About the Role

The **Junior Frontend Engineer (Level 1)** is an entry-level professional dedicated to building high-quality, performant, and visually stunning user interfaces. In this role, you will bridge the gap between design and implementation, working closely with UX/UI designers and backend engineers to deliver seamless digital experiences.

Your primary mission is to master modern frontend technologies while contributing to real-world production codebases. You are expected to be an "active learner," mastering not just the syntax of frameworks but the underlying principles of the web platform.

## Key Responsibilities

### 1\. UI Implementation & Styling

*   **Component Development:** Build and maintain reusable UI components using modern frameworks (React, Vue, or Angular) and TypeScript.
*   **Pixel-Perfect Execution:** Translate high-fidelity designs (Figma/Adobe XD) into responsive, accessible, and cross-browser compatible web pages.
*   **CSS Mastery:** Apply advanced CSS techniques (Flexbox, Grid, CSS Variables) and pre-processors/utility frameworks (Sass, Tailwind) to achieve polished layouts.

### 2\. Guided Technical Execution

*   **API Intergration:** Assist in connecting frontend interfaces with RESTful or GraphQL APIs, handling asynchronous data flow and state management.
*   **Quality Assurance:** Write unit and integration tests for UI components using tools like Jest, Vitest, or React Testing Library.
*   **Performance Awareness:** Learn and apply basic performance optimization techniques, such as lazy loading, image optimization, and code splitting.

### 3\. Team Integration & Growth

*   **Collaborative Learning:** Participate in pair programming and code reviews to learn best practices in frontend architecture and clean code.
*   **Documentation:** Maintain clear documentation for UI components and frontend workflows to ensure team alignment.
*   **Feedback Integration:** Actively seek feedback on code quality and UI implementation, iterating quickly to meet production standards.

## Success Measures

*   **UI Accuracy:** Consistency in delivering implementations that match design specifications with high fidelity.
*   **Code Quality:** Adherence to the project's linting rules, styling guides, and testing coverage requirements.
*   **Growth Velocity:** Demonstrated progress in mastering the specialized frontend tech stack and internal tooling.
*   **Responsiveness:** Proactive communication of roadblocks and timely delivery of assigned UI tasks.

## Core Competencies

*   **Web Fundamentals:** Strong understanding of HTML5 semantic tags, CSS3, and modern JavaScript (ES6+).
*   **Framework Foundation:** Basic experience with at least one modern frontend library/framework (e.g., React hooks, Vue Composition API).
*   **UI/UX Sensibility:** An eye for detail, spacing, typography, and smooth transitions.
*   **English Proficiency:** Upper-intermediate English (B2) for technical documentation and international collaboration.

## Requirements

*   Bachelor’s Degree in Computer Science, Web Development, or a related field (or equivalent self-taught portfolio).
*   0–2 years of experience (relevant internships, bootcamps, or personal projects).
*   Familiarity with Version Control (Git) and modern build tools (Vite, Webpack).
*   **A portfolio or GitHub repository showcasing clean, responsive frontend projects is mandatory.**
