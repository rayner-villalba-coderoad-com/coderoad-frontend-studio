# Intermediate Frontend Engineer (Level 2)

| Title: | Intermediate Frontend Engineer (Level 2) |
| --- | --- |
| Department: | Delivery / Frontend Engineering |
| Reports to: | Senior Frontend Engineer |
| Employment Type: | Contractor |

## About the Role

The **Intermediate Frontend Engineer (Level 2)** is a key contributor within our engineering PODs, responsible for the consistent delivery of high-quality frontend features and complex user interfaces. At Level 2, you move beyond foundational learning and are expected to handle well-defined technical challenges with increasing independence.

Your primary focus is **High-Quality Execution**. You will write clean, maintainable code using the latest frontend standards, participate actively in Agile ceremonies, and collaborate effectively with designers and backend engineers. While you will still receive guidance on architectural decisions, you are expected to take initiative in problem-solving and meeting sprint goals.

## Key Responsibilities

### 1. Technical Implementation
*   **Feature Development:** Lead the implementation of features and UI components using **React (Hooks, Context API)** and **TypeScript**, ensuring they are performant, accessible, and inclusive.
*   **System Maintenance:** Proactively identify and resolve complex bug fixes, ensuring the long-term health and stability of the frontend codebase.
*   **Modern Workflow:** Optimize development productivity using **Vite** and adhere to utility-first styling patterns with **Tailwind CSS**.
*   **Quality Ownership:** Implement comprehensive unit and integration tests (e.g., Vitest, React Testing Library) to validate your work before peer review.

### 2. Team Contribution & Collaboration
*   **Active Participation:** Engage deeply in Daily Stand-ups, Sprint Grooming, and Retrospectives, providing clear updates and identifying potential blockers early.
*   **Peer Review:** Contribute to the growth of the team by participating in code reviews, providing constructive feedback, and receiving reviews with a high degree of **coachability**.
*   **Shared Knowledge:** Contribute to technical documentation and component library READMEs to ensure the team's knowledge remains shared and accessible.

### 3. Continuous Learning & Process
*   **Skill Mastery:** Actively work to master the tools and technologies within the frontend ecosystem, seeking out mentorship to bridge gaps in architectural knowledge.
*   **Standard Adoption:** Consistently follow the team's Git workflows, CI/CD processes, and "Clean Code" principles (Sass, CSS Variables, DRY).

## Success Measures
*   **Predictability:** Ability to estimate frontend tasks accurately and deliver them reliably within the agreed-upon sprint timeframe.
*   **Technical Growth:** Demonstrable improvement in technical depth and the ability to handle increasingly complex UI logic over time.
*   **Quality Ratio:** Maintaining a low rate of "returned" tickets by prioritizing quality and validation steps during development.
*   **Team Synergy:** Positive contributions to the POD's velocity and a proactive attitude toward unblocking teammates.

## Core Competencies
*   **Technical Proficiency:** Professional experience with **React, TypeScript, and Tailwind CSS**. Strong logical reasoning for debugging complex system interactions.
*   **Proactive Communication:** A "can-do" attitude and the willingness to ask questions early to avoid roadblocks.
*   **Clean Code Mindset:** A strong commitment to writing readable, maintainable, and efficient code.
*   **English Proficiency:** Advanced English skills suitable for technical discussions and drafting clear documentation.

## Requirements
*   Bachelor’s Degree in Computer Science, Engineering, or a related field (or equivalent professional experience).
*   2–4 years of professional experience in frontend development.
*   Proven experience working within an Agile/Scrum framework.
*   **Portfolio Requirement:** A portfolio or GitHub profile demonstrating experience with React-based applications and modern styling libraries.

*   High-impact projects for international clients.
*   Holidays Off & Paid Time Off.
